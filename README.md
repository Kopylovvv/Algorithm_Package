# Алгоритмический пакет

Пакет, включающий в себя алгоритмы 6-ти сортировок,
поиска факториала и чисел фибоначи а также структуру данных

## Сортировки

## Факториал
Рекурсиваня и итеративная функции для вычисления факториала числа
### использование
```bash
python -m src.main factorial 5 
python -m src.main factorial 7 -r
```

## Числа Фиббоначи
Рекурсиваня и итеративная функции для вичисления чисел Фиббоначи
### использование
```bash
python -m src.main fibo 8
python -m src.main fibo 10 -r
```
